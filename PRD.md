# PRD — 2인 협력 텍스트 어드벤처 「령안 (ryeongan)」

## 1. 개요

현대 판타지 세계관을 배경으로 한 2인 실시간 협력 텍스트 어드벤처 게임. 심령현상·몬스터·헌터·퇴마가 공존하는 서울을 무대로, 두 플레이어가 각자 다른 정보를 보며 소통·협력하여 사건을 해결한다.

---

## 2. 세계관

- **배경**: 현재의 서울. 심령현상과 몬스터가 실재하지만 일반인에게 은폐된 세계.
- **조직**: 제도권 바깥의 민간 자경단 네트워크. 헌터·퇴마사·정보상 등 다양한 구성원이 느슨하게 연결.
- **글로벌 수치 — 심령 활동 수위**: 세계 전체의 긴장도를 나타내는 수치. 플레이어의 선택·실패·랜덤 인카운터 결과에 따라 오르내림. 임계점 도달 시 피날레 에피소드 강제 트리거.

---

## 3. 핵심 게임플레이 루프

```
랜덤 인카운터 발생
    → 선택 (스탯 또는 아이템으로 해금)
    → 결과 (스탯 변화 / 아이템 획득 / 심령 활동 수위 변동)

에피소드 트리거 조건 충족
    → 에피소드 진행 (구조화된 스토리 + 협력 퍼즐)
    → 결과 반영 (스탯·아이템 이월, 세계 상태 변화)

심령 활동 수위 임계점 도달
    → 피날레 에피소드 → 엔딩
```

---

## 4. 플레이어 시스템

### 4.1 스탯 체계

두 축으로 구성. 행동 스탯은 두 플레이어 공통 성장 가능, 감각/지식 스탯은 루팅을 통해 자연 분화.

| 축 | 설명 | 예시 스탯 |
|----|------|----------|
| 행동 스탯 | 무엇을 할 수 있는가 | 전투력, 탐색력, 교섭력 |
| 감각/지식 스탯 | 무엇을 인식·해석할 수 있는가 | 령안, 오컬트 지식, 심리 분석 |

- 스탯은 랜덤 인카운터 결과 및 에피소드 보상으로 성장.
- 특정 스탯 수치 이상이면 해당 선택지 해금.
- 클래스 없음 — 플레이어가 어떤 인카운터를 선택하느냐에 따라 자연스럽게 전문화.

### 4.2 아이템 체계

- 스탯과 완전히 독립적인 별개 시스템.
- 아이템 소지 또는 소모 시 특정 선택지 해금.
- **아이템은 개인 소유 고정. 플레이어 간 이전 없음.**
- 선택지 해금 조건이 아이템일 경우 두 플레이어 중 한 명이라도 보유하면 진행 가능 (OR 조건).
- 같은 선택지를 스탯으로도, 아이템으로도 해금 가능 — 플레이 스타일에 따른 다양한 경로 허용.

### 4.3 캐릭터 성장 구조

- 스탯·아이템은 에피소드 간 이월.
- 에피소드는 독립된 사건이지만 이전 선택이 NPC 태도·분기에 영향.
- 심령 활동 수위가 임계점에 도달하면 엔딩 강제 진입 — 무한 성장 불가.

---

## 5. 2인 협력 시스템

### 5.1 화면 동기화

| 상황 | A 화면 | B 화면 |
|------|--------|--------|
| 일반 진행 | 서술 텍스트 공유 + 각자 다른 선택지 목록 | 동일 서술 + 각자 다른 선택지 목록 |
| 핵심 협력 장면 | 서술 텍스트 자체가 다름 (예: 귀신의 말이 보임) | 다른 서술 텍스트 (예: 물리적 현상만 보임) |

- 선택지는 보유 스탯·아이템에 따라 각자에게 다르게 표시.
- 두 플레이어 모두 의미있는 선택지를 제공 — 쓸모없는 구경꾼 역할 없음.

### 5.2 협력 퍼즐

- 핵심 메커니즘: **정보 비대칭 + 스탯 비대칭**
  - A만 볼 수 있는 단서 + B만 할 수 있는 행동이 맞물려야 퍼즐 해결.
  - 채팅창을 통해 단서·상황을 공유하며 협력.
- 평소엔 같은 공간에서 진행, 클라이맥스·협력 퍼즐에서만 한시적 공간 분리.

### 5.3 채팅

- 게임 화면 내 실시간 채팅창 상시 제공.
- 단서 공유, 전략 논의, 선택 조율에 사용.

---

## 6. 에피소드 구조

### 6.1 랜덤 인카운터

- 에피소드와 무관하게 언제 발생해도 자연스러운 생활밀착형 사건.
  - 예: 길에서 이상한 물건을 주웠다 / 골목에서 수상한 소리가 들린다 / 뉴스에서 괴담이 흘러나온다.
- 결과: 스탯 변화 / 아이템 획득 / 심령 활동 수위 변동.
- 모든 콘텐츠 사전 제작 (런타임 LLM 없음).

### 6.2 에피소드

- 특정 조건(스탯 수치 / 아이템 보유 / 심령 활동 수위 / 스토리 플래그) 충족 시 트리거.
- 구조화된 스토리 + 분기 + 협력 퍼즐 포함.
- 이전 에피소드의 선택이 NPC 태도·분기에 영향.
- 실패(게임 오버) 시 해당 에피소드 처음부터 재시작. 스탯·아이템은 에피소드 진입 시점으로 복구.

### 6.3 피날레

- 심령 활동 수위 임계점 도달 시 강제 트리거.
- 엔딩 종류는 누적된 선택·세계 상태에 따라 분기.

---

## 7. 세션 관리

- **완전 실시간 강제**: 두 플레이어 모두 접속 중이어야만 진행.
- 한 명이 접속 종료 시 자동 일시정지 + 상태 저장.
- 두 플레이어가 재접속 시 저장 시점부터 재개.
- 혼자서는 어떠한 진행도 불가.

## 9. MVP 범위

### 포함

- 2인 실시간 WebSocket 동기화
- 각자 다른 선택지 렌더링 (스탯/아이템 기반)
- 채팅창
- 스탯 시스템 (행동 + 감각/지식)
- 아이템 시스템 (개인 소유, OR 조건 해금)
- 랜덤 인카운터 3종 (버스 정류장 유품 / 심야 골목 발소리 / 지하철 이상 승객)
- 에피소드 1: 「접선」
  - 두 플레이어가 서로 모르는 타인으로 시작, 민간 자경단 네트워크에 각자 섭외
  - 핵심 협력 퍼즐 1개 (정보 교환형: A단서 + B단서 → 채팅으로 맞춤)
  - 한시적 공간 분리 장면 1개 (합정동 폐상가 2층, `split`)
- 게임 오버 + 에피소드 재시작
- 세션 저장/재개

### 제외 (이후 확장)

- 심령 활동 수위 시스템 (구조만 설계)
- 피날레 에피소드
- 에피소드 2+
- 엔딩 분기

## 10. 에피소드 1 — 「접선」 (구현 완료)

**전제**: 두 플레이어는 서로 모르는 타인. 민간 자경단 네트워크의 누군가에게 각자 따로 연락을 받고 같은 장소에 나타난다.

**목표**: 세계관 소개 + 첫 협력 퍼즐 체험 + 파트너 관계 형성

**구조** (구현 기준):
1. 소집 — 각자 다른 경위로 약속 장소 도착 (A/B 개별 서술, 병렬 도입부)
2. 첫 만남 + 조직 소개 (공유 서술)
3. 임무 브리핑 — 합정동 폐상가의 심령 현상 조사
4. 건물 진입 → **핵심 협력 퍼즐 (2층 복도, 공간 분리 `split`)**: A에게만 보이는 단서 + B에게만 보이는 단서 → 채팅으로 공유 → 정답 도출
5. 서버 전용 분기 판정(`branch_check`)으로 결과 분기 — 완전 성공 / 부분 성공 / 실패
6. 결과별 에필로그 + 스탯/아이템 획득, 실패 시 에피소드 진입 체크포인트에서 재시작

> 콘텐츠 정의: `src/content/data/episodes/ep1.json` (+ 시나리오 `ep1.scenario.md`)
